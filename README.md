# E-Commerce Sales Dashboard (Power BI)

## 🔹 Project Overview

This project is an interactive Power BI dashboard built using the Brazilian Olist e-commerce dataset.

## 🔹 Key Features

* KPI tracking (Total Sales, Orders, Freight)
* Sales trend analysis over time
* Top-performing product categories
* Interactive filters for dynamic analysis

## 🔹 Tools Used

* Power BI
* DAX
* Data Modeling

## 🔹 Insights

* A few categories contribute the majority of revenue
* Sales show a steady trend over time
* Freight cost is a significant expense component

## 🔹 Files Included

* `.pbix` Power BI file
* Dataset files
